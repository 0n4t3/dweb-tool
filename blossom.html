Blossom

Fetch a file from Blossom servers (blob storage used alongside Nostr) by its SHA-256 hash. The tool asks every configured server whether it holds the blob, downloads it — from several servers in parallel when possible — and verifies the result against the hash locally, so you never have to trust an individual server.
